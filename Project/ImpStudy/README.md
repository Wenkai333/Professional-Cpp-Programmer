# ImpStudy

A C++ project created from template.

## Building

```bash
./scripts/build.sh
```

## Running

```bash
./build/src/main
```

## Testing

```bash
./scripts/test.sh
```

## Project Structure

- `src/` - Source code
- `tests/` - Test files
- `docs/` - Documentation
- `scripts/` - Build and utility scripts
